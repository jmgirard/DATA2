Challenge Question

  1. Look at the code below and figure out how it works.

  2. Adapt this code to create four new variables called carat_minimum, carat_maximum, price_minimum, and price_maximum.

diamonds |> 
  summarize(
    across(
      .cols = c(carat, price),
      .fns = list(m = mean, s = sd),
      .names = "{.fn}_{.col}"
    )
  )
# A tibble: 1 × 4
  m_carat s_carat m_price s_price
    <dbl>   <dbl>   <dbl>   <dbl>
1   0.798   0.474   3933.   3989.

Click here for the answer key 
diamonds |> 
  summarize(
    across(
      .cols = c(carat, price),
      .fns = list(minimum = min, maximum = max),
      .names = "{.col}_{.fn}"
    )
  )
# A tibble: 1 × 4
