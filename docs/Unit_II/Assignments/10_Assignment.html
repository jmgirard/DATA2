Question 1

  1. Download the records.csv file from Canvas, put it in your project folder, and import it into R. Examine the tibble to familiarize yourself with it; it contains world records for several running events (as of October 2020).

  2. Add a new variable to the tibble called kmph that contains the average speed attained in each record in kilometers per hour (you will need to calculate this from existing variables - be careful about units). Save this change so that you can use this new variable in future questions.

Question 2

  1. Create a summary table using your data from Question 1 that contains the minimum, mean, and maximum speeds (in kmph).

  2. Create a summary table using your data from Question 1 that contains the minimum time (in seconds) and maximum speed (in kmph) for each distance (in meters).
